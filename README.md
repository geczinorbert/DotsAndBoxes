# DotsAndBoxes
# Usage: 

  ## First step is to set the path for java like: 
    set path=%path%;C:\Program Files\Java\jdk1.8.0_181\bin
  ## Second step is to run the program like: 
    java DotsAndBoxes -d 10 
## Game description
 The game starts with an empty grid of dots. The player and the computer take turns adding a single horizontal or vertical line between two unjoined adjacent dots. A player who completes the fourth side of a 1Ã—1 box earns one point and takes another turn. The game ends when no more lines can be placed. The winner is the one with the most points.
